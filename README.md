# Image resizer

## Techstack
- HTML + Tailwind + vanilla js
- Nodejs
- Redis

## How to run
- Install [docker](https://docs.docker.com/engine/install/) and [docker-compose](https://docs.docker.com/compose/install/)
- In terminal `docker compose up -d`
- Access via port 53798
